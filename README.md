# 2021년 1학기 아롬 7기 멘토링 📗

앱 개발의 기초를 배우고 싶은 신입 부원들을 대상으로 멘토링을 진행합니다.

### 📝 진행 방식 
해당하는 주마다 과제가 이슈를 통해 등록됩니다.  
참고 자료와 강의를 활용하여 과제를 수행하고, 과제 파일을 Pull requests 하시면 됩니다.  
다 해결하지 못했더라도 진행과정만큼 제출하시면 됩니다.
